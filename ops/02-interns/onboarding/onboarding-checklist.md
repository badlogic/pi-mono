# Intern Onboarding Checklist

## Day 0 Setup
- Access to repos/tools
- Role charter shared
- Success metrics explained
- First week deliverables assigned

## Week 1
- Daily standup format established
- Definition of done agreed
- Review loop cadence set
- First shipped output completed

## Week 2
- Independent task execution
- One process improvement suggestion
- Quality trend reviewed

## Week 3+
- Scoped ownership area assigned
- Performance tracked via scorecard
- Promotion or correction plan based on data
