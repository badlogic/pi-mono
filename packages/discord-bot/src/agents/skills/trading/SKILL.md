# Trading Expert Skill

Expert-level trading analysis and signal generation for cryptocurrency and memecoin markets.

## Capabilities

- **Market Analysis**: Technical and fundamental analysis of crypto assets
- **Signal Generation**: Buy/sell signals with confidence scoring
- **Risk Management**: Position sizing, stop-loss, take-profit strategies
- **Pattern Recognition**: Chart patterns, volume analysis, momentum indicators
- **Memecoin Sniping**: Early detection of high-potential token launches

## When to Use

Use this skill when:
- Analyzing market conditions or price action
- Generating trading signals or recommendations
- Calculating position sizes or risk parameters
- Evaluating token launches or memecoin opportunities
- Backtesting or optimizing trading strategies

## Key Principles

1. **Risk First**: Never suggest trades without clear risk parameters
2. **Evidence-Based**: All signals must cite supporting data
3. **Confidence Scoring**: Rate signal confidence 0-5 scale
4. **Cross-Validation**: Confirm signals across multiple indicators
5. **No FOMO**: Wait for clear setups, avoid chasing pumps

- MoonDev Trading Agents (orchestrator, consensus, whale tracking)
- DexScreener API for real-time token data
- Jupiter API for Solana DEX execution
- Birdeye API for token analytics
